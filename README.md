# Εγκατάσταση.

## Από φάκελο κινητού.

Υπάρχει ο φάκελος termux στο κινητό κινητού. Εκτελούμε:

```
termux-setup-storage
pkg install nnn
```
Με το nnn βρίσκουμε το install.sh και το εκτελούμε.

## Από git.

Εκτελούμε:

```
pkg install git
git clone https://github.com/tsekaris/termux
sh ~/termux/install.sh
```
